# SGPD - Sistema de Gestão de Processos Digitais

Protótipo frontend de um SGPD com aparência enterprise SaaS, em HTML/CSS/JS puro.

## Como usar

Abra `index.html` em um navegador moderno.

## Telas incluídas

- Dashboard principal com sidebar, header, cards e tabela de atividades.
- Formulário de abertura de solicitação com anexos e ações.
- Painel de triagem e delegação com tabela, filtros e modal.
